Overview
I led the development of a comprehensive web application designed to manage the entire process from order creation to delivery, seamlessly integrating with customers' existing SAP systems. The project utilized Blazor for building a dynamic, interactive user interface, leveraging C# for both client-side and server-side development. The application provided a unified platform where users could create orders, track inventory, manage shipments, and monitor deliveries in real-time.

The software is used for internal tracking of the current status of the order, which includes the stock of the items used for that order. It consists of multiple user logins, which provides custom tailor-made views depending on the role of the user.

A key aspect of the project involved integrating with customers' SAP systems to ensure data synchronization and consistency. This integration allowed the application to fetch real-time data on inventory levels, customer information, and order statuses directly from SAP, ensuring that all stakeholders had access to accurate and up-to-date information. I implemented robust data exchange protocols, including secure APIs and data transformation layers, to facilitate smooth communication between the web application and SAP systems.

Throughout the project, I was responsible for the entire software development lifecycle, from initial requirement gathering and architecture design to coding, testing, and deployment. I collaborated closely with stakeholders to refine the application's functionality and ensure it met business needs. The final solution not only streamlined operations and improved efficiency but also provided detailed analytics and reporting features, enabling better decision-making. This project demonstrated my ability to develop and integrate complex systems, ensuring seamless data flow and operational efficiency.
